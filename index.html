About This Site

This site is part of the Cloud Resume Challenge. The goal of the challenge is to build a resume site hosted entirely on AWS using infrastructure-as-code and cloud-native services, while tracking visitors in a serverless and scalable manner.

Architecture Overview

The architecture follows the steps described in the Cloud Resume Challenge:

  • Frontend hosted on Amazon S3 and served via CloudFront for global caching and SSL.
  • Visitor counter implemented using AWS Lambda functions and API Gateway for the API.
  • Visitor data stored in DynamoDB with monitoring via CloudWatch.
  • Infrastructure is defined as CloudFormation templates and deployed using CI/CD pipelines.
